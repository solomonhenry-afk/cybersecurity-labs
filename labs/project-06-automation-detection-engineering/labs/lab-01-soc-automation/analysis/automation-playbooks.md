# SOC Automation Playbooks

## Purpose
Automate repetitive SOC tasks to allow analysts to focus on investigation and response.

## Playbook 1 — Alert Intake
- Normalize alert fields
- Validate required metadata
- Assign initial severity

## Playbook 2 — Enrichment
- Pull asset inventory data
- Query threat intelligence context
- Attach business impact tags

## Playbook 3 — Decision Routing
- Low-risk → auto-close with justification
- Medium-risk → analyst review
- High-risk → escalation + containment workflow

## Impact
Automation removes human bottlenecks while preserving accountability and auditability.
