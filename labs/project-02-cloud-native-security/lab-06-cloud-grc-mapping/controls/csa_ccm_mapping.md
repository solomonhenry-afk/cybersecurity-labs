# CSA Cloud Controls Matrix (CCM) Mapping

## IAM — Identity & Access Management
- **IAM-01**: Identity lifecycle management
- **IAM-12**: Privileged access restrictions

Mapped to Azure Entra ID and AKS identities.

---

## IVS — Infrastructure & Virtualization Security
- **IVS-09**: Secure configuration management
- **IVS-06**: Hypervisor and container isolation

Mapped to Kubernetes baseline and runtime protections.

---

## LOG — Logging & Monitoring
- **LOG-01**: Log collection
- **LOG-04**: Event correlation

Mapped to Defender for Cloud and CNAPP telemetry.

---

## SEF — Security Incident Management
- **SEF-02**: Incident response processes
- **SEF-05**: Root cause analysis

Mapped to attack path correlation and runtime alerts.

---

## Summary
CSA CCM bridges technical cloud controls with
cloud service provider governance requirements.
